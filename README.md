# Fitzsixto's Portfolio

Personal portfolio website built with SvelteKit and TypeScript.

## Tech Stack

- SvelteKit
- TypeScript
- Tailwind CSS
- Vercel

## Features

- Responsive design
- Interactive canvas cursor
- Project showcase with live demos
- Professional experience timeline
- Skills overview
